# Results

The results follow a similar directory structure as the 
[run_args](../run_args) directory. Please refer to the
directory structure there to understand how the results
are categorized.

The predictions can be found in `predictions.json` files
in the corresponding run e.g. `run_4` in each variation's
`runs` directory. The `pred_metrics.json` provides the
$\text{mAP@1}$, $\text{mAP@D}$ and structural metrics.
